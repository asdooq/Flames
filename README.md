# FLAMES Love Calculator 💘

A modern, animated web application that calculates relationship dynamics based on the classic FLAMES game (Friends, Love, Affection, Marriage, Enemies, Siblings).

## Features

- **Classic Algorithm**: Accurately calculates the relationship outcome by removing common letters between two names.
- **Beautiful UI**: Features a modern glassmorphism design with a responsive layout.
- **Dynamic Animations**: Includes floating background particles, pop-in text effects, and smooth page transitions.
- **Theme Changes**: The background colors and particle effects dynamically change based on the result (e.g., fiery colors for Enemies, lovely pinks for Love).
- **Easter Eggs**: Secret results for specific names.

## Technologies Used

- **HTML5**: Structure of the application.
- **CSS3**: Styling, layout, and animations (Vanilla CSS, no external frameworks).
- **JavaScript**: Logic for the FLAMES algorithm, theme switching, and UI transitions.

## How to Use

1. Clone or download the repository.
2. Open the `index.html` file in any modern web browser.
3. Enter your name in the first input field.
4. Enter your crush's name in the second input field.
5. Click the **Find Relationship** button to see your result!

## Note on Privacy

This application contains a hidden form that logs submitted names via Formspree for fun/analytical purposes as implemented by the original creator.
